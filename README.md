<h1>Expense Tracker</h1>
A modern and intuitive Expense Tracker Web Application built using ReactJS and Apache ECharts to help users efficiently manage and visualize their monthly spending.

<h3>Overview</h3>

This project allows users to track daily expenses, categorize spending, and gain insights through interactive pie chart visualizations. It focuses on a clean UI, smooth user experience, and meaningful financial insights.

<h3>Features</h3>

* ➕ Add and manage expenses with ease
* 📅 Filter expenses by month
* 🗂️ Categorize spending (Food, Travel, Bills, etc.)
* 📊 Interactive pie chart visualization of expenses
* 💾 Persistent data using LocalStorage
* 📱 Responsive and user-friendly interface

<h3>Problem It Solves</h3>

Managing daily expenses manually can be confusing and inefficient. This application provides:

- A centralized platform to log expenses
- A visual breakdown of spending habits
- Better awareness for financial decision-making
  
<h3>Tech Stack</h3>

 - **Frontend**: ReactJS (Vite)
 
 - **Data Visualization**: Apache ECharts
   
 - **Styling**: CSS, Tailwind
   
 - **Storage**: Browser LocalStorage

<h3>Visualization</h3>

The application uses Apache ECharts to display:

 - Category-wise expense distribution
 - Percentage contribution of each category
 - Interactive and dynamic chart updates
